# ✨ Scroll & Scribble

A modern blogging platform where users can register, authenticate, create blogs, edit their own content, and explore posts from other users.

Built as a full-stack MERN project.

---

## 🚀 Live Demo

Frontend:

https://blog-frontend-mauve-alpha.vercel.app/

Backend API:

https://scroll-scribble-backend.onrender.com

---

## ✨ Features

- Secure Login & Registration
- JWT Authentication
- Create Blogs
- Read Blogs
- Edit Own Blogs
- Delete Own Blogs
- Responsive Design
- Protected Routes
- Owner/Admin Authorization
- Beautiful Dark UI

---

## ⚙️ Installation

Clone repository

```bash
git clone https://github.com/shauryasync/blog-frontend
```

Install dependencies

```bash
npm install
```

Create `.env`

```
VITE_API_URL=YOUR_RENDER_BACKEND/api
```

Run project

```bash
npm run dev
```
